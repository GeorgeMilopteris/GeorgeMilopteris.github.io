Project Idea

Project Idea: A Survival Game

Summary: My Project will be a first person survival video game which would be played on PC. This game would be centred around PvP (Player vs Player) although there would be a few PvE (Player vs Entity) aspects to the game. This game would be set on an island with players starting with nothing and either surviving on their own or teaming up with a few friends in order to survive. The main focus of this game would be to get players to collect and craft items and progress from for example bows to assault rifles. Additionally players would need to build bases in order to store their loot, however players will be able to raid/break other players bases in order to take their loot. There would be a system in order for players to be able to progress, which would make players have to find higher tier or mid tier items by any means even if that means through pvp, then have to research these items in order to be able to craft them, themselves.

Motivation: With there being many survival games available to play at the moment, with many being PvP focussed I don’t believe any of them fully utilise or correctly execute many exciting aspects/mechanics available to survival games. So this game would expand on those mechanics providing survival game enthusiasts with the best experience by refining each aspect. Additionally because I primarily play survival games i’ve realised that a strong line of communication with the developers and the community can lead to both good and bad in terms of updates, therefore another aspect of this project would be to allow for strong user input, however this would not take over the main development of this project.

Description: This project will be a survival game played off PCs and would be able to be purchased from Steam (preferably), this game would cost $15 (USD).
This game would be played from the first perspective of your character, players would be randomly generated a character model when they launch the game, with every player having different but similar character models. Although through playing the game players will find and craft clothes in order to survive and customize the look of their characters, as some clothes will primarily be for cosmetic looks and others purely for defense (armour).
As the game would be played from first-person it would be a first-person shooter as will developing through the players will need to be able to have weapons such as guns in order to combat other players in PvP. This will most likely form the basis of the game as players are forced to leave their bases to collect items they are susceptible to being killed and losing their items they have gathered.
Therefore players are forced to build bases to store their loot/items from other players there would be a building system in place for players to be able to be creative and unique in their base design. This flexibility of building would then allow players to create bases in which are difficult to raid/break and so they can defend it from other players.
Therefore as building is a primary feature of this game then raiding is equally as important, although there would be no competition between players. So there would be a meticulous raiding system in place which allow players at all stages of the game to be able raid others. For example once players have reached the later stages and have the best weapons and armour they will most likely be able to raid others with rockets, explosives and more modern forms of explosives whereas players early on will be forced to raid with tools and primitive explosives such as hand made grenades etc.
Similarly players will have access to various weapons starting at melee weapons such as swords and primitive weapons such as muskets and even bows and crossbows. Then as players progress they would get access to more modern weapons such as bolt rifles, ak-47’s M4’s, homemade rifles and more variants of different guns ranging from full automatic to semi automatic. This is to ensure players have a wide variety of choice and preference to guns they would like to use and play with.
To increase/encourage PvP there would certain locations across the map which building is restricted and is where better tier loot spawns for players to be able to loot and try their luck. This would therefore cause players to fight of these items hence, these locations would also range from more basic ones and more modern/higher tier ones.

Tools and Technologies: This game would run on a Unity engine, and would require an office or group of developers to have PCs in order to constantly update and provide a quality experience. Players would all require internet to download and play this game as it would be 100% played online versus other players. This would most likely require the developers to use various applications such as github to share to and experiment with code, as well due to githubs flexibility and accessibility.

Skills Required: Firstly, there would need to be a development dedicated to creating this project, who all have extensive knowledge and skill with Unity and game development. At this current point in time with my knowledge and access to such hardware I do not believe that it would be possible for me to develop such a project. However in the future, finding and acquire these skills would take some time studying and developing and would then be a potential career path/ feasible option.

Outcome: If this project were to ever be successful, then it would not only make my life more enjoyable but others as well who have been looking for a more complete and solid survival game, to enjoy and entertain themselves. Depending on the success of this game it could also cause a shift in many survival games to then focus on improving their game to be able to keep their player base resulting in more quality games for everyone to enjoy.
